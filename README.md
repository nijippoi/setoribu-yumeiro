# セトリ部ツイート支援ツール (夢色ビジョン)

## 概要

[夢色ビジョン](https://twitter.com/yumeiro_tw)のセトリ部のツイート支援ツールです。[こちら](https://nijippoi.github.io/setoribu-yumeiro/)のGitHubページにて最新のバージョンを公開しています。モバイルのブラウザからホームスクリーンに追加できます。
